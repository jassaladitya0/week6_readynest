**NexusData AI Studio** is an enterprise-grade client-side Data Intelligence & Analytics Platform designed and developed by **Aditya Jassal**. It enables seamless dataset upload, interactive data cleaning, visual charting, in-memory SQL query execution, statistical modeling, correlation matrix heatmaps, AI predictions, and complete multi-chart PDF report exports.

---

## ✨ Core Features & Capabilities

### 1. 📥 Dataset Upload Landing Page (Initial View)
- Drag-and-drop or browse local CSV, Excel (`.xlsx`, `.xls`), and JSON files.
- Instant Schema Profiler: Automatically infers column data types, row counts, missing values, and cleanliness health scores.

### 2. 📊 Executive Dashboard & AI Intelligence
- **Created by Aditya Jassal** executive interface.
- Real-time KPI summaries, AI Predictions & Anomaly Highlights.
- **Pearson Correlation Matrix Heatmap**: Interactive matrix grid for numeric features.

### 3. 📈 Multi-Chart Visual Showcase (6+ Visualizations)
- 📊 **Bar Chart**: Categorical Performance & Aggregation.
- 📈 **Line Chart**: Time-series & Distribution Trends.
- 📉 **Area Chart**: Cumulative Distribution Analysis.
- 🥧 **Pie Chart**: Segment Market Share.
- 🔴 **Scatter Plot**: Dual Metric Correlation.
- 🕸️ **Radar Chart**: Multi-Variable Metrics Profile.

### 4. 📄 Complete Multi-Page PDF Report Generator
- Generates a PDF report containing:
  - **Executive Summary & Key Performance Indicators (KPIs)**.
  - **AI Predictions & Anomaly Detection Findings**.
  - **Numeric Statistical Analysis Summary Table**.
  - **Pearson Correlation Matrix Heatmap**.
  - **Visual Chart Showcase (All Chart Types Embedded)**.
  - **Author Credit Footer**: `Developed by Aditya Jassal`.

### 5. 🧹 Data Wrangler, SQL & Stats Workbench
- Full client-side SQL execution via `AlaSQL`.
- Data cleaning (deduplication, missing value imputation, text casing, type casting).
- Pivot Table Engine and Exploratory Data Analysis.
